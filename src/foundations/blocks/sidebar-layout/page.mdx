A full application shell: a collapsible sidebar with icon navigation, a top bar with
actions, and a scrollable content area. Toggle the sidebar from the top bar. It composes
[Avatar](/ui/avatar), [Button](/ui/button) / `IconButton`, and [Divider](/ui/divider),
plus Phosphor icons.

Use the viewport toggles above the preview to see how it adapts from desktop down to
mobile, or open it full-screen.

## Best Practices

1. **State**: the collapsed and active-item state is local `useState` — lift it to your router so the active item reflects the current route.
2. **Navigation**: swap the `NAV_ITEMS` buttons for your router's links (e.g. `asChild` with an `<a>`), keeping the `data-active` styling.
3. **Responsiveness**: below `md`, collapse the sidebar by default and overlay it as a [Drawer](/ui/drawer) rather than shrinking to icons.
