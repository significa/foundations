## Features

- **Keyboard Navigation**: Use arrow keys to navigate between tabs.
- **Automatic Activation**: Tabs are automatically activated when selected.
- **ARIA Support**: Includes proper ARIA attributes for accessibility.
- **Orientation Support**: Supports both horizontal and vertical layouts.
- **Controlled & Uncontrolled**: Can be used in both controlled and uncontrolled modes.
