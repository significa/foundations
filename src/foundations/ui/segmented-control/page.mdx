## Features

- **Smooth Animations**: Animated indicator transitions using Motion layout animations
- **Keyboard Navigation**: Full keyboard support with arrow key navigation
- **Controlled & Uncontrolled**: Flexible state management options
- **ARIA Support**: Full accessibility implementation with tab role
- **Custom Styling**: Extensible styling through className props

  <SegmentedControl.Item />

## Accessibility

The Segmented Control component implements the [WAI-ARIA Tabs Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/tabs/). Each item has `role="tab"` and `aria-selected` reflects the current selection state. Arrow keys move focus and selection between segments.
