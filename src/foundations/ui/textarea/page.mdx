### Auto-resize

Add the `field-sizing-content` class to grow the textarea to fit its content.
Pair it with a `max-h-*` (and `overflow-auto`) class to cap the growth. Browsers
without support fall back to a fixed-height textarea.

<Preview slug="textarea-resize" />

## Best Practices

1. **Sizing**:
   - Consider initial height based on expected content
   - Allow resizing when appropriate
   - Maintain consistent width with other inputs

2. **Accessibility**:
   - Provide clear labels
   - Consider character/word count indicators
