## API Reference

The hooks returns a boolean, if the user has set a preference to reduce motion animations e.g. transition animations

## Best Practices

To enhance accessibility, ensure this is set when using animations to comply with WCAG guidelines. If this setting is enabled on the user's computer, it may cause UI issues.
