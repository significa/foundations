A grouped bar chart for comparing discrete categories. Each category gets a band on
the x axis (`scaleBand`), and every series in `config` is drawn as a bar within that
band. Built on plain SVG with no dependencies; series colours come from the
[Chart Container](/charts/chart-container) palette.

## API Reference

Each key in `config` (other than `xKey`) renders as a grouped bar. See
[Chart Container](/charts/chart-container) for the `ChartConfig` shape.

## Best Practices

1. **Grouping**: two or three series per category stays readable; beyond that prefer small multiples.
2. **Theming**: reference colours via `var(--color-<key>)` from the `--color-chart-*` palette.
3. **Baseline**: bars must start at zero — do not crop the y axis.
