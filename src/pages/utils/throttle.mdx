# Throttle

A utility which ensures that a given function is only executed at most once during a specified time interval. This is useful for limiting the frequency of function calls in scenarios such as handling scrolling, resizing, or other high-frequency events.

## Install

<FileCode path="lib/utils/throttle.ts" />

## API Reference

**Parameters:**

- `func` (function): The function to throttle.
- `intervalMs` (number): The time in milliseconds to wait before allowing the next call of the function.

**Returns:**

- Returns a throttled version of the original function that will only execute once per specified time interval.

## Usage

```ts copy
import { throttle } from '[path]/throttle';
```

```ts copy
const throttledFunction = throttle(() => {
  // do stuff
}, 1000);
```

## Examples

### Scroll Measurement

```ts copy
import { throttle } from '@/lib/utils/throttle';

const scrollToTopButton = document.querySelector('#scroll-to-top');

function toggleScrollToTopButton() {
  const shouldDisplay = window.scrollY > window.innerHeight;

  scrollToTopButton.style.display = shouldDisplay ? 'block' : 'none';
}

const throttledScroll = throttle(toggleScrollToTopButton, 512);

window.addEventListener('scroll', throttledScroll);
```
