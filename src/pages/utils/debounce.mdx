# Debounce

A utility which ensures that a given function is only executed after a specified delay period has passed since the last time it was invoked. This is useful for limiting the frequency of function calls in scenarios such as handling user input, resizing events, or other high-frequency actions.

## API Reference

**Parameters:**

- `func` (function): The function to debounce.
- `intervalMs` (number): The time in milliseconds to wait after the last call before invoking the function.

**Returns:**

- Returns a debounced version of the original function that will only execute once after the specified delay, unless interrupted by further calls within the delay window.

## Usage

```ts copy
import { debounce } from '[path]/debounce';
```

```ts copy
const debouncedFunction = debounce(() => {
  // do stuff
}, 256);
```

## Examples

### API Call

```ts copy
import { debounce } from 'lib/utils/debounce';

function dummyApiCall(query: string) {
  console.log(`Searching for: ${query}`);
}

const searchInput = document.querySelector('#search');
const debouncedSearch = debounce(dummyApiCall, 300);

searchInput.addEventListener('input', (event) => {
  const query = (event.target as HTMLInputElement).value;

  debouncedSearch(query);
});
```
