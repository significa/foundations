# Checkbox

Checkbox component with support for an indeterminate state.

<Box>
  <Checkbox />
</Box>

## Install

<FileSource path="@/foundations/components/Checkbox" />

### Dependencies

```bash copy
npm install @radix-ui/react-checkbox
```

## API Reference

This component is built on top of `@radix-ui/react-checkbox`. For more information and API reference, check [their documentation](https://www.radix-ui.com/primitives/docs/components/checkbox#root).

## Usage

```tsx copy
import { Checkbox } from '[path]/Checkbox';
```

```tsx copy
<Checkbox />
```
