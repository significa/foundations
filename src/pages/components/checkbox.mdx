## Install

<FileCode path="components/foundations/Checkbox" />

### Dependencies

```bash copy

```

This component is built on top of `@radix-ui/react-checkbox`. For more information and API reference, check [their documentation](https://www.radix-ui.com/primitives/docs/components/checkbox#root).

## Usage

```tsx copy
import { Checkbox } from '[path]/Checkbox';
```

```tsx copy
<Checkbox />
```
