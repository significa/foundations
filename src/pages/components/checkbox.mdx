## API Reference

| Prop             |   Type    | Default Value |
| :--------------- | :-------: | ------------: |
| `required`       | `boolean` |             - |
| `defaultChecked` | `boolean` |             - |
| `checked`        | `boolean` |             - |
| `disabled`       | `boolean` |             - |
| `intermediate`   | `boolean` |             - |
| `name`           | `string`  |             - |
| `value`          | `string`  |             - |

| Data attribute    |        Values         |
| :---------------- | :-------------------: |
| `[data-disabled]` | Present when disabled |
