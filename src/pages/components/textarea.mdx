# Textarea

A multiline text input field for user entry that supports labels.

<PreviewBox>
  <Textarea rows="12" placeholder="Tell us something." />
</PreviewBox>

## Install

<FileCode path="components/foundations/Textarea" />

## API Reference

| Prop           |         Type          | Default Value |
| :------------- | :-------------------: | ------------: |
| `label`        |       `string`        |             - |
| `placeholder`  |       `string`        |             - |
| `name`         |       `string`        |             - |
| `value`        |       `string`        |             - |
| `defaultValue` |       `string`        |             - |
| `error`        |       `boolean`       |             - |
| `disabled`     |       `boolean`       |             - |
| `required`     |       `boolean`       |             - |
| `rows`         | `number` \| `string`  |             - |
| `resizable`    | `boolean` \| `string` |             - |

## Usage

```tsx copy
import { Textarea } from '[path]/Textarea';
```

```tsx copy
<Textarea />
```
