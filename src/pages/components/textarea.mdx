# Textarea

A multiline text input field for user entry that supports labels.

<Box>
  <Textarea rows={12} placeholder="Tell us something." />
</Box>

## Install

<FileSource path="@/foundations/components/Textarea" />

## API Reference

| Prop           |         Type          | Default Value |
| :------------- | :-------------------: | ------------: |
| `label`        |       `string`        |             - |
| `placeholder`  |       `string`        |             - |
| `name`         |       `string`        |             - |
| `value`        |       `string`        |             - |
| `defaultValue` |       `string`        |             - |
| `error`        |       `boolean`       |             - |
| `disabled`     |       `boolean`       |             - |
| `required`     |       `boolean`       |             - |
| `rows`         |       `number`        |             - |
| `resizable`    | `boolean` \| `string` |             - |

## Usage

```tsx copy
import { Textarea } from '[path]/Textarea';
```

```tsx copy
<Textarea />
```
