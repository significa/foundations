## API Reference

| Prop           |    Type     | Default Value |
| :------------- | :---------: | ------------: |
| `type`         |  `string`   |             - |
| `label`        |  `string`   |             - |
| `placeholder`  |  `string`   |             - |
| `name`         |  `string`   |             - |
| `value`        |  `string`   |             - |
| `defaultValue` |  `string`   |             - |
| `error`        |  `boolean`  |             - |
| `disabled`     |  `boolean`  |             - |
| `required`     |  `boolean`  |             - |
| `icon`         | `ReactNode` |             - |

## Usage

```tsx copy
import { Input } from '[path]/Input';
```

```tsx copy
<Input type="..." />
```
