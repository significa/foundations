# Input

A flexible type-agnostic input field supporting labels and custom icons.

<FileCode path="components/foundations/Input" />

## Usage

```tsx copy
import { Input } from '[path]/Input';
```

```tsx copy
<Input type="..." />
```

## API Reference

| Prop          |    Type     | Default Value |
| :------------ | :---------: | ------------: |
| `type`        |  `string`   |             - |
| `label`       |  `string`   |             - |
| `placeholder` |  `string`   |             - |
| `icon`        | `ReactNode` |             - |
| `error`       |  `boolean`  |             - |
| `disabled`    |  `boolean`  |             - |
