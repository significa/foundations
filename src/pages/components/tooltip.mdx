## API Reference

| Prop             |                                                                                Type                                                                                | Default Value |
| :--------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------: | ------------: |
| `placement`      | `top` \| `right` \| `bottom` \| `left` \| `top-start` \| `top-end` \| `bottom-start` \| `bottom-end` \| `left-start` \| `left-end` \| `right-start` \| `right-end` |         `top` |
| `persistOnClick` |                                                                             `boolean`                                                                              |       `false` |
| `delayIn`        |                                                                              `number`                                                                              |           `0` |
| `delayOut`       |                                                                              `number`                                                                              |           `0` |

## Usage

```tsx copy
import { Tooltip } from '[path]/Tooltip';
```

```tsx copy
<Tooltip content="Hello world">
  <p>Hover me</p>
</Tooltip>
```
