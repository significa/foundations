## Install

<FileCode path="components/foundations/Marquee" />

Add the keyframes and css animation plugin to your `tailwind.config.cjs`.

```js copy filename="tailwind.config.cjs"
module.exports = {
  theme: {
    extend: {
      keyframes: {
        marquee: {
          '0%': { transform: 'translateX(0%)' },
          '100%': { transform: 'translateX(-100%)' }
        }
      }
    }
  },
  plugins: [require('tailwindcss-animate')]
};
```

### Dependencies

```bash copy
npm install -D tailwindcss-animate
```
