# useViewport

Custom hook that provides the current viewport size and optional resize handling with debouncing.

## Install

<FileSource path="@/foundations/hooks/useViewport" />

## API Reference

- **Parameters:**

  - `options` _(optional)_: An object with optional configuration parameters:
    - `debounce` _(boolean | number)_: If `true`, applies a default debounce interval of 32ms to the resize event. If a number, specifies the debounce interval in milliseconds. If omitted, no debounce is applied.
    - `onResize` _(function)_: A callback function that receives the updated viewport size whenever the viewport is resized. If this function is provided, the hook will not update the `size` state directly and will rely on this function to handle size updates.

- **Returns:**
  - An object with the current `width` and `height` of the viewport.

---

## Usage

```tsx copy
import { useViewport } from '[path]/useViewport';

function Component() {
  const { width, height } = useViewport();

  // ...
}
```
