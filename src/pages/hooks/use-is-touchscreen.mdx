# useIsTouchscreen

Custom hook that detects whether the user's device has a touchscreen.

## Install

<FileSource path="@/foundations/hooks/useIsTouchscreen" />

## API Reference

- **Returns:**
  - Returns a boolean indicating whether the device is a touchscreen. It returns `true` if a touchscreen is detected, otherwise it returns `false`.

## Usage

```tsx copy
import { useIsTouchscreen } from '[path]/useIsTouchscreen';

function Component() {
  const isTouchscreen = useIsTouchscreen();

  // ...
}
```
