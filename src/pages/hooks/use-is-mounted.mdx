# useIsMounted

Custom hook that returns a function to check if a component is currently mounted.

## Install

<FileSource path="@/foundations/hooks/useIsMounted" />

## API Reference

- **Returns:**
  - A state that is `true` if the component is currently mounted and `false` if it hasn't been mounted.

---

## Usage

```tsx copy
import { useIsMounted } from '[path]/useIsMounted';

function Component() {
  const isMounted = useIsMounted();

  // ...
}
```
