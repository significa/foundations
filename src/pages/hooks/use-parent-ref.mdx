# useParentRef

Custom hook that returns a ref to the parent element of a given child ref.

## Install

<FileSource path="@/foundations/hooks/useParentRef" />

## API Reference

- **Parameters:**

  - `childRef`: A mutable ref object pointing to the child DOM element whose parent is to be tracked.

- **Returns:**
  - Returns a mutable ref object that holds the latest parent element of the specified child ref. Returns `null` if the child ref is not set or has no parent.

---

## Usage

```tsx copy
import { useParentRef } from '[path]/useParentRef';

function Component() {
  const childRef = useRef<HTMLDivElement | null>(null);
  const parentRef = useParentRef(childRef);

  // ...
}
```
